# 我的待办清单

一个简洁的任务管理工具，帮你记录和追踪日常待办事项。

## 功能

- 添加新任务
- 标记任务完成
- 删除任务
- 任务数据自动保存（刷新不丢失）

## 在线体验

访问：todo.preview.tencent-zeabur.cn

## 技术栈

- HTML
- CSS
- JavaScript
- localStorage（本地存储）
